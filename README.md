Flashy – Language Flashcard App

A simple desktop flashcard application built with Python Tkinter and pandas to help users learn foreign language vocabulary through spaced repetition.

Features:

Random flashcard generation
Flip cards to reveal translations
Tracks known words automatically
Saves progress to words_to_learn.csv
Continues from where you left off
Simple and intuitive GUI
How It Works

The app loads vocabulary from french_words.csv. As you mark words as known, they are removed from the learning list and saved to words_to_learn.csv, ensuring personalized learning progress over time.

Controls:

✔️ Right button → mark word as known
❌ Wrong button → skip to next card
Click card → flip between languages

Tech Stack:

Python
Tkinter (GUI)
pandas (data handling)
